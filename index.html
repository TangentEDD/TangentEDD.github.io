Machine Learning-Based Home Purchase Behavior Prediction (2026)

Problem: Predict potential homebuyers for a real estate platform to improve marketing ROI, addressing data imbalance and multicollinearity.

Data:200,000 global property transaction records with 25 features (income, crime rate, property age, etc.).

Approach: Data preprocessing (IQR capping, log transform, feature engineering: EMI/income ratio, price per sqft). Built 9 models: MLP, Random Forest, LightGBM, XGBoost, etc. Grid search + 10-fold CV.

Outcome:MLP achieved AUC 0.95, Recall 99.4%, F1 0.854. EMI/income ratio (67%) and crime rate were top factors. Business value: Enables precise targeting of high-intent buyers, reducing wasted ad spend and improving marketing efficiency. Provides data foundation for personalized loan product recommendations.

Contribution:Co-programmer: implemented Random Forest, LightGBM, and XGBoost models; tuned hyperparameters; contributed to GitHub documentation.

Outcome: Agents learned basic jumping strategies and achieved stable average scores. The environment provided a reproducible testbed for RL algorithms and demonstrated how reward shaping influences agent behavior.

Contribution: Built the environment class, coded physics and reward logic, implemented rendering and play mode, and ran experiments with different RL algorithms. Code available on GitHub.
